# Ration

 Ration is a web application that allows users to manage and track their daily nutrient intake. The app provides information about various ingredients and recipes, as well as allows users to create and store their own recipes and more!
<br>
<br>
<br>

### Home Page
#### At the home page users must log in to gain full access to features. 
#### Users can log in/out and access other pages via the nav bar at the top of the page.
#### Additionally the nav bar at the top contains the ability to search.
#### Users have limited access until logged in.

#### Here users have the ability to view a list of their recipes.
